# mikaeleramian.com

Personal portfolio built as a Windows XP desktop in the browser.

Live at [mikaeleramian.com](https://mikaeleramian.com)

## Stack

- React + Vite
- XP.css
- react-draggable
- react-resizable

## Features

- Windows XP desktop UI
- Draggable/resizable windows
- Boot sequence
- Taskbar with minimize/restore
- Start menu
